# Ruta PM: skills para tu trabajo de producto

Una ruta opcional para product managers. Usa exactamente lo que aprendiste en el módulo 3 (skills) — pero en vez de aplicarlo a un sitio personal, lo aplicas a tu trabajo real: discovery, PRDs, historias de usuario, y métricas de éxito.

**Al terminar tendrás:** cuatro skills propias — reutilizables en tu trabajo desde mañana — y los primeros documentos de una idea de producto tuya (real o hipotética): insights de discovery, un PRD, historias de usuario, y métricas de éxito.

## Lecciones

1. **Tu idea de producto** — una idea real (o inventada) que va a atravesar toda la ruta.
2. **Descubrimiento** — una skill que convierte notas de entrevistas en insights estructurados.
3. **De idea a PRD** — una skill que convierte una idea en un PRD borrador.
4. **Historias de usuario** — una skill que convierte un PRD en historias de usuario.
5. **Métricas de éxito** — una skill que define cómo sabrás si la idea funcionó.

**Necesitas:** haber terminado el módulo 3 (ya sabes construir una skill).

**Tiempo estimado:** 45–60 minutos. Puedes parar cuando quieras y volver con `/continuar`.

Para empezar, escribe: `/ruta-pm`
